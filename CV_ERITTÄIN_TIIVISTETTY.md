# Digital Detective - CV Tiivistelmä (Erittäin Tiivistetty)

## Teknologiat

**Kielet:** Python 3

**Kirjastot:** BeautifulSoup4, Requests, Rich, python-dotenv, argparse, pandas

**API:t:** SerpAPI, IP-API, IP-Location API

**Työkalut:** Sherlock, Social Analyzer

## Taidot

- Web scraping (BeautifulSoup)
- API-integraatiot ja HTTP-pyynnöt
- CLI-työkalun kehitys
- Caching ja rate limiting
- Data validation ja ristiinvertailu
- Error handling
- JSON-käsittely
- Regex
- Subprocess-hallinta
- Unit testing
- Git ja versiohallinta

## Projektin Kuvaus

OSINT-työkalu Pythonilla, joka kerää julkista tietoa henkilönimistä, IP-osoitteista ja käyttäjänimistä useista API:sta. Sisältää web scrapingin, API-integraatiot, cachingin, datan validointia ja modulaarisen arkkitehtuurin.
